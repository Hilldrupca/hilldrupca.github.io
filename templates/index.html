Projects:

  • Applied Data Science with Python

    All of my work for the Coursera specialization of the same name. The following python modules were used: matplotlib, networkx, nltk, pandas, and scikit-learn.

  • Data Structures and Beyond Capstone Project

    A java capstone project for a Coursera specialization of the same name. Explores the modeling of information cascades through a graph. A real world example of such a case would be the sudden increase in popularity of a new trend.

  • Github Page

    This website. If you like the layout, you are welcome to clone the repository, and make changes to suit your needs.

  • LeetCode

    My submissions for the Monthly Challenge card problems, and various other card exercises. Most also have accompanying unit tests.

  • Nutrition Calculator

    A meal planner that uses data provided by the USDA National Nutrient Database for Standard Reference. Currently uses SR28 (2015) data. I've also ported the project to a Flask application running on the Google Cloud Platform.
